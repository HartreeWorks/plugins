# HartreeWorks Plugins

Claude Code plugins by [Peter Hartree](https://x.com/peterhartree).

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add hartreeworks/plugins
```

Then install any plugin:

```
/plugin install plugin-name@hartreeworks-plugins
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| [project-management](https://github.com/HartreeWorks/plugin--project-management) | Manage non-coding projects with scaffolding, Google Docs integration, and memory tracking |
